## Project Overview

* This project is a backend-focused URL shortening service built to understand how real-world systems handle URL generation, redirection, and data persistence.
* The goal was not just to build a working application, but to explore design decisions and trade-offs involved in read-heavy systems.

## Core Features

- Short URL generation and redirection
- Persistent storage of original and shortened URLs
- Basic collision handling strategy
- Clean REST API design

## Tech Stack

- Backend: Node.js, Express
- Database: mySQL
- API Style: REST

## Running Locally

1. Clone the repository
2. Install dependencies
3. Set environment variables
4. Start the server
